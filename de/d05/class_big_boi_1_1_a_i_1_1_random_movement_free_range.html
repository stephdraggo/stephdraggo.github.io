Detailed Description

Generates random targets within a range determined by mesh renderer. Tested with plane and cube, rounded objects will most likely be treated as cubes when calculating area bounds. Tickbox to allow vertical movement or keep entities grounded. Mesh renderer must exist but can be disabled. Limitations: area bounds are set at start and cannot be modified in runtime.
